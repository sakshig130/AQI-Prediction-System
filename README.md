# AQI Prediction System

A web-based Air Quality Index (AQI) prediction and monitoring system developed for Dhanbad. The application uses pollution and weather data to predict air quality levels and provide users with useful environmental insights.

## Features

- Real-time AQI monitoring
- AQI prediction using Machine Learning
- Weather forecast integration
- Health advisories based on AQI levels
- Pollution trend analysis
- Nearby hospital information
- Interactive dashboard

## Technologies Used

- Python
- Flask / FastAPI
- XGBoost
- HTML
- CSS
- JavaScript
- REST APIs

## Project Structure

```
api/
data/
models/
src/
start.py
requirements.txt
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/sakshig130/AQI-Prediction-System.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python start.py
```

## Dashboard

The dashboard displays:

- Current AQI
- Predicted AQI
- Weather information
